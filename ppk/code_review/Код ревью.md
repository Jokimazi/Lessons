Ошибки: 
1. Имя файла должно называться так же, как и главный класс файла
2. У каждого метода должны быть аргументы, нельзя использовать внешние переменные внутри методов
3. ```List<Order> orders = new List<Orders>``` можно упростить до
   ```List<Order> orders = []```
4. Поменять `int choice = int.Parse(Console.ReadLine());` на `string choice = Console.ReadLine();` и заменить интовые значения на стринговые в меню, чтобы не было ошибки несоответствия типов при вводе не числа, следовательно, и вылета программы
5. Перед объявлениями типами данных в получениях данных из консоли следует поставить - "?", что будет означать, что для переменной будет доступно значение Null, что в свою очередь просит IDE в случае если значение из консоли будет Null, что правда невозможно, так как данные полученные из консоли автоматически приводиться к стрингу
